## Context

GitNexus precomputes dependency graphs (symbols, call edges, clusters, execution flows) at index time using tree-sitter + KuzuDB. This replaces multi-turn grep-read-grep cycles with single-query tool responses. The 7 MCP tools (query, context, impact, detect_changes, rename, cypher, list_repos) give coding agents decision-ready context before writing code.

Installed globally and verified on EPYC repos: 92K nodes, 150K edges, 2,230 clusters, 868 flows across 4 repos.
