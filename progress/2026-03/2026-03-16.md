### Why

GitNexus precomputes dependency graphs at index time (tree-sitter → KuzuDB property graph → Leiden clustering → execution flow detection). This gives coding agents dependency-aware context in single-query tool responses instead of multi-turn grep cycles. The 7 MCP tools (query, context, impact, detect_changes, rename, cypher, list_repos) are available via `gitnexus mcp` or CLI.
